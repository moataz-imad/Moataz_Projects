Technical Stuff

  • The frontend of the webpage is designed and developed using HTML, CSS, and Javascript.
  • The backend is developed using Python, Flask, and Javascript
  • The numerical method used to find the root was the Bisection Method. Although it was a powerful way to find the root, it had some disadvantages so I had to develop my own (basic) method of optimizing the sectioning and finding the root.
  • The algorithm will try its best to find up to 2 roots in the given range, although it might face some difficulties in some conditions. emojies where used to represent the difficulty of finding the root
